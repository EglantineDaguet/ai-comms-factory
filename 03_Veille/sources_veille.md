## Comment utiliser ces sources

Pour un scan actu hebdomadaire :

1. Parcourir d'abord les newsletters daily/weekly (Rundown AI, Marion Cas, Import AI)
2. Vérifier les blogs officiels des éditeurs IA pour les annonces produit
3. Survoler La Réclame et Blog du Modérateur pour l'actu communication
4. Repérer dans les médias internationaux les analyses de fond qui peuvent nourrir un post Pilier 3

Pour un sujet précis, croiser au moins deux sources de catégories différentes (un éditeur officiel + un média indépendant + une analyse de fond) avant de prendre position.
